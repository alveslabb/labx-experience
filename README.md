#  Sistema de Gerenciamento de Laboratório Escolar

## 📌 Descrição
Este projeto tem como objetivo desenvolver um sistema para organizar o uso do laboratório da escola de forma eficiente.

Este projeto consiste no desenvolvimento de um sistema para organizar e gerenciar o uso do laboratório escolar de forma eficiente e planejada, pois, muitas vezes o laboratorio acaba ficando de lado e inutilizado. A ideia principal é garantir que cada turma tenha acesso ao laboratório uma vez por semana, com um cronograma bem definido em que cada dia seja destinado a uma turma específica.

---

## 🎯 Objetivo
Criar um sistema simples e funcional que:
- Organize o cronograma de uso do laboratório  
- Amentar a ultilização do laboratório
- Incentive aulas práticas  

---

## ⚙️ Funcionalidades
-  Cadastro de turmas  
-  Definição de dias da semana para cada sala  
-  Organização automática do uso semanal  
-  Acesso para professores visualizarem horários  
-  Registro de atividades realizadas no laboratório  

---

## 👥 Usuários do Sistema

### Professores
- Consultar horários disponíveis  
- Ver o dia reservado para sua turma  
- Registrar atividades realizadas  

###  Alunos
- Visualizar quando terão aula no laboratório  
- Acompanhar atividades práticas  

###  Direção / Administração
- Gerenciar turmas  
- Organizar cronograma semanal  
- Controlar o uso do laboratório  

---

## 📆 Funcionamento do Sistema
- Cada turma terá **um dia fixo na semana**
- O sistema distribui as turmas ao longo dos dias
- Evita que duas turmas utilizem o laboratório no mesmo horário
